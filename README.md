# Lancr

A mobile app for freelancers to manage clients, projects, time tracking, and invoicing.

## Features

- Client and project management
- Built-in timer for tracking work hours
- Invoice generation with PDF export
- Data backup and restore
- Dark/light theme support

## Tech

- React Native with Expo SDK 54
- TypeScript
- expo-sqlite for local storage
- React Navigation

## Getting Started

```
npm install
npx expo start
```

## Build APK

```
eas build --platform android --profile preview
```
